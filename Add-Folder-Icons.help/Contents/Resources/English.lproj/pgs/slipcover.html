SlipCover Integration

Managing SlipCover styles

SlipCover from Bohemian Coding is a program that lets you make custom case icons for media files, such as home-made CDs, DVDs, Blu-Ray discs and so-on. It comes with various case designs built in, you can download more from the Internet or even design your own from scratch.

Add Folder Icons lets you define styles which use SlipCover case designs, thanks to code generously donated by the SlipCover author. For more about this and other contributions, please see the "About" window, accessed from the "About Add Folder Icons" menu entry in the "Add Folder Icons" menu.

If Add Folder Icons detects the presence of any SlipCover case designs which it hasn't seen before, it will ask you if you want to automatically create styles for each of the case designs it has found. Please note that SlipCover must be installed directly in your Applications folder, rather than in any subdirectory of Applications or any other folder, in order to be detected.


New SlipCover Styles Found

You can also choose to add SlipCover based styles by hand at any time - go to the Edit Style panel and select "SlipCover cases" from the "Generate folder icons using:" menu at the top.


Editing a SlipCover Style

The menu underneath will be enabled and you can choose the case design from there.

When you use SlipCover styles, Add Folder Icons uses pictures it considers to be cover art for the thumbnail, picking the first one that it finds. You can control how cover art is chosen using the advanced preferences.

It is not possible to use a mixture of SlipCover and custom style parameters; you must choose one or the other.

If you ever delete a SlipCover case design which has an associated Add Folder Icons style, Add Folder Icons will be forced to delete that style as it is no longer valid. It will warn you when this happens.
