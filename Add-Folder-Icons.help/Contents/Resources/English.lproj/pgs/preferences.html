Preferences

Change the way Add Folder Icons works.


General Settings

The General settings are reasonably self-explanatory. You can choose the style applied to folders added to the folder list by default. The "Add Sub-Folders Too" menu setting can be toggled from this window as well. Finally, you can decide whether the folder list is cleared after icons are successfully added, or if the contents are kept.


Advanced Settings

In the Advanced settings you control how Add Folder Icons looks for cover art, when it is using a style which specifies that only cover art pictures are to be used in an icon. Use the and buttons to add and remove entries from the table. Each entry gives the name of a file, without any filename extension, which will be treated as cover art. So any kind of picture will be used, provided it matches one of the names in the list. The check box underneath provides an additional way to specify cover art — if the option is checked, assigning any colour label to the picture using the Finder will cause the picture to be treated as cover art.
