Managing Styles

Design your own styles

To create and edit styles, click on the button below the folder list and select "Manage Styles..." or choose the "Manage Styles..." entry in the Window menu.


Opening the Manage Styles Window

You will see a list of currently defined styles.


The Manage Styles Window

Some of these are built-in presets. They can't be altered or removed. Everything else is up to you! Add a style with the button, remove a style selected in the list with the button and edit a style selected in the list by clicking on the button and choosing "Edit Style..." from the menu that pops up. The style editor panel will open.


The Style Editor Panel

A preview image gives you an idea of how your final image will look. If your screen is big enough, you can use the resize handle in the bottom right of the panel to make it larger and get a better view of the icon, all the way up to the maximum 512x512 pixels supported size.

If you select the "Search for a single cover or box art image" option, then rather than looking for all pictures in a folder and choosing some at random, Add Folder Icons looks for files it considers to be cover art and picks the first that it discovers. Normally, this means any picture with a filename of "cover" or "folder" (in a mixture of upper or lower case letters and with any filename extension), or any file which has a colour label applied to it in the Finder regardless of its name. You can change how cover art is chosen using the advanced preferences.

The "Save Style" button saves the new style, or any changes you may have made to an existing style. It becomes available immediately for use in the folder list. To discard changes or throw away a new style, use the "Cancel" button instead.
